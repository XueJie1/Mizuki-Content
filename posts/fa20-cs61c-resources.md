## Labs and Projects 实验和项目

You can fork from the repositories of [61c-teach](https://github.com/61c-teach).

你可以从[61c-teach](https://github.com/61c-teach)的仓库复刻。

[fa20-lab-starter](https://github.com/61c-teach/fa20-lab-starter)

The repository names of all projects follow the format of `fa20-projN-starter`. You can search for `fa20-proj` on [61c-teach's GitHub homepage](https://github.com/61c-teach) to find all project repositories.

项目的仓库名都遵守`fa20-projN-starter`的格式，在[61c-teach的 GitHub 主页](https://github.com/61c-teach)搜索`fa20-proj`即可搜到所有项目仓库。
